# 365 Dias

Aplicação web minimalista que representa visualmente cada dia do ano, mostrando em tempo real o progresso anual, a data atual e o horário.

## 🌐 Acesse o projeto

👉 https://cidosjr.github.io/365-dias/

---

## ✨ Funcionalidades

- Um ponto representa cada dia do ano
- Dias passados ficam destacados
- Destaque para o dia atual
- Relógio em tempo real
- Data atual automática
- Reinício automático a cada novo ano
- Interface minimalista e responsiva

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
